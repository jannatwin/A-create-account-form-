# 📋 Interactive Account Registration Form with Real-Time Validation


A beautiful, modern, and highly interactive user registration interface featuring robust client-side validation. Built entirely with Vanilla HTML5, CSS3, and JavaScript, it incorporates glassmorphism, fluid interactive card lift effects, and instant error feedback loops to ensure a smooth user experience.

---

## ✨ Features

   Real-Time Client-Side Validation:** Validates field inputs dynamically on submission.
    Username:* Ensures the field is not empty.
    Email:* Verifies the field is populated and matches a strict regex email pattern.
    Password:* Verifies presence of a password.
    Password Check:* Verifies presence and guarantees it matches the initial password input.
   Premium Glassmorphic UI:** Features a translucent registration card, subtle drop shadows, and high-contrast gold-colored titles over an elegant background image backdrop.
   Fluid Micro-Animations:** Clean `:hover` transitions and input field transforms that gently lift cards (`translateY`) to enhance visual hierarchy.
   Instant UI State Transitions:** Visual validation feedback (green border for success, red border/warning message display for failures) seamlessly triggered via CSS class changes.
   Responsive & Accessible Design:** Centered viewport layout optimized for different displays.
